This README would normally document whatever steps are necessary to get the

Things you may want to cover:

Install ruby 2.3.0:

  rvm install 2.3.0

Install bundler:

  gem install bundler

Install gems:

  bundle install

Setup databases:

  bundle exec rake db:setup

Setup sample apps:

  bundle exec rake db:seed

Running specs:

  bundle exec rspec
